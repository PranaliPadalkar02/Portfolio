Pranali's Cookbook Website Project

For this project, I created a cookbook website to showcase some of my favorite recipes. This project allowed me to apply my HTML and CSS skills to design an aesthetically pleasing and user-friendly website. Additionally, I integrated Google Tag Manager (GTM) to gain insights into user behavior and interactions on the site.

Technologies Used:

  • HTML: Structured the content, creating an intuitive navigation flow for users.
  • CSS: Styled the website to make it visually appealing, applying color schemes, fonts, and layouts.
  • Google Tag Manager: Implemented an empty GTM container on all pages to facilitate future analytics tracking and insights.
  • GitHub: Uploaded the project to a GitHub repository, making it accessible for version control and collaboration.
  • Netlify: Deployed the project on Netlify for easy accessibility and sharing.

Project Structure:

  • Homepage: Designed a clean and welcoming homepage, providing an overview of the cookbook's content.
  • Recipe Page: Created a dedicated page for at least one featured recipe, presenting detailed instructions and ingredients.
  • Contact Us: Implemented an interactive and user-friendly "Contact Us" section, providing visitors with a convenient way to reach out, share feedback, or ask questions. The section features a thoughtfully designed form that allows users to input their name, email, and message. This form not only enhances user engagement but also serves as a valuable tool for building connections with the community.
